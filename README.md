# gob minimalist agent

## Computer as THE tool 
- built in archlinux environment
- native file system 
- can maintain and evolve own codebase
- grows and evolves to suit user needs

## Quick Start

```bash
git clone https://github.com/dusty-schmidt/gob-01.git
cd gob-01
pip install -e .
python -m gob.main --tui
```
